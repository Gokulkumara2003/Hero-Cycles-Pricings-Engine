# Hero-Cycles-Pricings-Engine
 Hero Cycles Pricing Engine:  A full-stack Spring Boot application developed to manage Hero Cycles configurations, components, and pricing. The application enables sales teams to manage cycle parts, configure cycles, and instantly calculate price breakdowns based on selected components.

 Features

- Component Management (Create, Read, Update, Delete)
- Cycle Management (Create, Read, Update, Delete)
- Assign Components to Cycles
- Dynamic Price Calculation
- Price Breakdown for Each Cycle
- Responsive User Interface
- REST APIs
- PostgreSQL Database Integration
- Interactive Dashboard
